1.PROJECT TITLE
Basic Password Strength Checker
2. Aim of the Project
The aim of this project is to create a Basic Password Strength Checker using C language that checks whether a password entered by the user is valid 
or not based on certain conditions.
3. Problem Statement
In todayâ€™s digital world, password security is very important.
This project checks the strength of a password by verifying:

Presence of at least one digit
Presence of at least one uppercase letter

5. Language and Tools Used
Programming Language: C
Compiler: GCC / Turbo C
Platform: Windows / Linux
6. Concepts Used
Character arrays (Strings)
Loops (for)
Conditional statements (if-else)
User-defined functions
String traversal
7. Algorithm
Start
Ask the user to enter a password
Read the password using scanf()
Pass the password to validation function
Check minimum length of password
Check if at least one digit is present
Check if at least one uppercase letter is present
If all conditions are satisfied, password is valid
Else password is invalid
Stop
8. Program Logic Explanation
The password is stored in a character array
A function validatePassword() is created
strlen() is used to check length
Loop is used to check each character
isdigit() checks digit
isupper() checks uppercase letter
Function returns success or failure
