# Basic Password Strength Checker

## 1. Project Title
**Basic Password Strength Checker**

---

## 2. Aim of the Project
The aim of this project is to create a Basic Password Strength Checker using C language that checks whether a password entered by the user is valid or not based on certain conditions.

---

## 3. Problem Statement
In todayâ€™s digital world, password security is very important.  
This project checks the strength of a password by verifying:
- Minimum length of password  
- Presence of at least one digit  
- Presence of at least one uppercase letter  

---

## 4. Why This Project Is Created
This project is created to:
- Understand string handling in C  
- Learn password validation logic  
- Practice loops, conditions, and functions  
- Get an idea of how real login systems validate passwords  

---

## 5. Language and Tools Used
- **Programming Language:** C  
- **Compiler:** GCC / Turbo C  
- **Platform:** Windows / Linux  

---

## 6. Concepts Used
- Character arrays (Strings)  
- Loops (`for`)  
- Conditional statements (`if-else`)  
- User-defined functions  
- String traversal  

---

## 7. Algorithm
1. Start  
2. Ask the user to enter a password  
3. Read the password using `scanf()`  
4. Pass the password to validation function  
5. Check minimum length of password  
6. Check if at least one digit is present  
7. Check if at least one uppercase letter is present  
8. If all conditions are satisfied, password is valid  
9. Else password is invalid  
10. Stop  

---

## 8. Program Logic Explanation
- The password is stored in a character array  
- A function `validatePassword()` is created  
- `strlen()` is used to check length  
- Loop is used to check each character  
- `isdigit()` checks digit  
- `isupper()` checks uppercase letter  
- Function returns success or failure
